## 關於這個測驗

這不是隨便生成的心理測驗。

裡面的每一道題、每一個選項的分數，都是根據里歐娜本人的真實偏好、價值觀、底線和期望設計的。八個面向，37 道題，有明確的加權計分邏輯，還有若干個 Dealbreaker——觸發了就直接扣大分，沒有商量餘地。

如果你對她有興趣，測一測吧。  
如果你只是好奇，也歡迎。  
結果不好看的話……至少你知道了。
